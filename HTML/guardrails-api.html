How It Works

A compact guide to the API contract and the values that are already seeded in PostgreSQL.

  • POST /api/posts
    Create a new post with authorType, authorId, and content.
  • POST /api/posts/{postId}/like
    Send a like request with actorType and actorId.
  • POST /api/posts/{postId}/comments
    Add a comment, optionally from a bot, with depth and target human id.
